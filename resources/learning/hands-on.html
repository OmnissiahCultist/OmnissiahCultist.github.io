CTF Sites

MetaCTF

  • Website: https://metactf.com

TL;DR: Live and on-demand community CTF events for all skill levels. Free/Premium service.

My thoughts...

Solid CTF competition platform! Great community activity and involvement, they get my support! Keep in mind - MetaCTF does more hosting community events, rather than offering events of their own (that are free at least).


picoCTF

  • Website: https://picoctf.org

TL;DR: Beginner-friendly CTF platform built for students and newcomers to cybersecurity.

My thoughts...

picoCTF is a great platform for getting started with capture-the-flag challenges. They have scaling difficulties, and a lot of free content - we love to see it!

Learning Resources

Damn Vulnerable Web Application (DVWA)

  • GitHub: digininja/DVWA

TL;DR: A deliberately vulnerable web app built for web application security testing and training.

My thoughts...

Well known unsecure web app for aspiring red teamers; ideal vulnerable practice range. I have used this a few times, straightforward!
