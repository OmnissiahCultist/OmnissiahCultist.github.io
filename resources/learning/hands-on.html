CTF Sites

MetaCTF

  • Website: https://metactf.com

TL;DR: Live and on-demand CTF events for all skill levels — ideal for collaborative play.

My thoughts...

MetaCTF has been one of the best CTF platforms I've used for events and team-based play. They offer good guidance and a friendly UI.


picoCTF

  • Website: https://picoctf.org

TL;DR: Beginner-friendly CTF platform built for students and newcomers to cybersecurity.

My thoughts...

picoCTF is a great platform for getting started with capture-the-flag challenges. It teaches concepts progressively, which is ideal for learners.



Lab Repositories

Antisyphon Lab Repo

  • GitHub: anti-syphon/labs

TL;DR: Hands-on labs that match Antisyphon courses — self-paced and practical.

Learning Resources

Damn Vulnerable Web Application (DVWA)

  • GitHub: digininja/DVWA

TL;DR: A deliberately vulnerable web app built for web application security testing and training.

My thoughts...

DVWA is a long-standing favorite for training in OWASP-style web vulnerabilities. Great for both red and blue teams.
